# 테이블 오더 시스탬 Qbit 서비스 기획서

## 🧾 프로젝트 개요
프로젝트 큐빗은 기존의 불편했던 메뉴판과 키오스크를 대체할 QR주문을 제시합니다. 멀리 붙어있는 메뉴판, 테이블당 한두개 있는 메뉴판, 줄서서 주문해야하는 키오스크, 이제 큐빗을 이용해 기다림과 불편함 없이 QR스캔으로 여러명이 같이 메뉴판을 보며 주문합니다. 큐빗은 동시에 메뉴판을 보며 서로의 메뉴를 확인하며 주문이 가능합니다.

## 01. 기획의도, 브리프, 목표, 타겟

📍 기획의도 
- QBIT은 오프라인 매장에서 발생하는 대기 시간, 불편한 메뉴 접근성, 주문 병목 현상을 해결하고자 기획되었습니다.<br />
- QR코드를 활용해 누구나 빠르고 쉽게 주문할 수 있는 시스템을 통해, 고객의 편의성과 점주의 운영 효율성을 동시에 향상시키는 것을 목표로 합니다.<br />

🎯 목표
- 테이블에서 바로 주문이 가능한 비대면 서비스 제공<br />
- 여러 사용자가 동시에 메뉴를 보고 주문할 수 있는 환경 구현<br />
- 키오스크 및 메뉴판의 한계를 보완하여 사용자 경험 향상<br />

👥 타겟 

- 소규모 음식점 및 카페 운영자<br />
- QR 기반 비대면 주문 시스템을 도입하고자 하는 점주<br />
- 모바일을 통한 간편한 주문을 선호하는 20~40대 소비자층<br />

브랜드 로고 컬러 및 포인트 컬러는 코랄 오렌지(#f76f5a)와 민트 그린(#4cdfa2)의 조합으로 눈에 띄면서 밝고 친근한 이미지를 표현했습니다.<br />

## 03. 기능
저희 Qbit 플랫폼은 QR코드 기반의 간편한 비대면 주문을 지원하며, 회원과 점주 모두에게 효율적인 경험을 제공합니다. 주요 기능은 다음과 같습니다 
- 간편 QR 주문: QR코드를 통해 빠르고 쉽게 주문할 수 있으며, 비대면으로 안전하게 이용 가능합니다.
- 실시간 공동 주문: 회원과 상대방이 각각 장바구니에 담아도, 실시간 메뉴 선택 상황을 확인할 수 있으며, 한쪽에서 주문하면 상대방도 동시에 주문됩니다.
- 주문 내역 즉시 확인: 주문 완료 후 즉시 확인 가능하여 편리하고 투명한 관리가 가능합니다.
- 점주 관리자 페이지: 점주를 위한 주문 관리 기능이 개발 중이며, 향후 더 편리한 운영 환경을 제공할 예정입니다.

## 04. 비즈니스 모델
Qbit은 QR코드 기반 테이블 오더 시스템을 통해, 회원이 스마트폰으로 간편하게 주문하고 결제할 수 있는 경험을 제공합니다.<br />
동시에 점주는 주문 현황을 실시간으로 관리하며, 효율적인 운영 환경을 구축할 수 있습니다.

## 05. 경쟁사이트 및 레퍼런스
한국에서도 테이블 오더 시스템을 제공하는 다양한 서비스가 있습니다. <br />
예를 들어, “Catchtable”은 고객이 테이블에 비치된 QR코드를 스캔하여 메뉴를 확인하고 주문 및 결제까지 할 수 있는 시스템을 제공합니다. <br />
또한, Catchtable은 비대면 주문과 실시간 주문 확인 기능을 통해 레스토랑 운영의 효율성을 높이는 서비스로 알려져 있습니다.<br />
이러한 경쟁 서비스들은 각기 다른 기능과 편의성을 제공하며, Qbit의 서비스를 개발할 때 참고 자료로 활용하였습니다.

## 06. 디자인 컨셉
전체적인 깔끔한 느낌과 곳곳에 배치된 포인트 색상의 화면 구성으로 누구나 쉽게 메뉴를 확인할 수 있으며, 명확한 버튼과 메뉴 구조로 원하는 메뉴를 빠르고 편리하게 선택할 수 있습니다.<br />
최대한 간단하게 디자인하여 남녀노소 누구나 쉽게 이용할 수 있도록 구성했습니다.<br />

## 08. 사이트 맵
```
메인 페이지
├─ 주문 내역 확인
├─ 장바구니 확인
│  ├─ 본인/상대방 메뉴 확인
│  ├─ 사이드 메뉴 포함
│  ├─ 수량 변경
│  └─ 메뉴 추가
└─ 메뉴 섹션 (스크롤 이동)

세부 메뉴 페이지
├─ 옵션 선택 (사이즈, 음료)
└─ 장바구니 담기

주문 완료 페이지
├─ 주문 완료 확인 + 주문 내역 확인 가능
└─ 자동으로 장바구니 비움

관리자 페이지 (추후 추가 예정)
├─ 사이드 메뉴 및 선택지 커스텀
└─ 로그인 / 회원가입
```
